# Changelog

All notable changes to Woodchuck are documented here.

## [1.1.0.0] - 2026-07-17

### Added

- Put the exact benchmark prompt directly in the opening section.
- Browse every published pirate summary and haiku in responsive, swipeable carousels.
- Rebuild and verify carousel data from the published responses in CI.

## [1.0.0.0] - 2026-07-17

### Added

- Publish the reproducible woodchuck model benchmark, deterministic rubric, and 71 successful responses.
- Explore model and reasoning-effort results through an AI Friday-styled static dashboard.
- Test every push and deploy the public site through GitHub Actions.

### Changed

- Separate publishable results from raw local run artifacts and provider metadata.
